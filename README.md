## 📌 Descripción del Proyecto

Este proyecto tiene como objetivo construir un modelo de **Machine Learning supervisado** para predecir si un posible cliente será **buen pagador** o **mal pagador**.

El desarrollo se realiza en **Databricks**, utilizando datos sintéticos de aproximadamente **20,000 registros**, con variables demográficas, laborales, financieras y crediticias.

El flujo incluye:

- Generación de datos sintéticos.
- Simulación de errores comunes en datos reales.
- Proceso ETL de limpieza y transformación.
- Preparación de variables para Machine Learning.
- Entrenamiento de un modelo supervisado.
- Evaluación del modelo.
- Predicción de nuevos clientes.

## 🎯 Objetivo General

Desarrollar un modelo predictivo que permita clasificar clientes según su probabilidad de cumplimiento de pago, apoyando la evaluación de riesgo crediticio.

---

## 🎯 Objetivos Específicos

- Generar una base sintética de clientes.
- Simular problemas comunes en los datos, como nulos, duplicados y valores extremos.
- Realizar limpieza y transformación mediante ETL.
- Entrenar un modelo supervisado con Árbol de Decisión.
- Evaluar el rendimiento del modelo mediante métricas de clasificación.
- Interpretar las predicciones obtenidas.
- Proponer mejoras para futuros modelos.
